◆ make_sahooks()

template<IoHandler IoHandlerT>
SAHooks shpxx::io::make_sahooks ( )

Create SAHooks object wrapping some type which provides the implementation.

The hooks produced will attempt to call static functions on the type passed as the template parameter. The hooks produced will convert to/from the specific typedefs used in the SAFile struct.

SHPOpenLL will use the SAHooks pointer directly, or copy the contents of the SAHooks struct; but it does not copy the pointer itself, so passing the pointer to a stack var is acceptable

Example:

auto hooks = make_sahooks<some_type_t>();
return SHPOpenLL("a_file", "rb+", &hooks);
